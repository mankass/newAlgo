# 💡 TIPS

### 🚀 Запуск
```bash
./gradlew run
```
Впиши вызов нужной функции в `main()` в `Main.kt`.

### ➕ Новая задача
Добавь функцию в файл по теме: `topics/arrays.kt`, `strings.kt`, `stack.kt` или `math.kt`.

### 🧪 Тесты
```bash
./gradlew test
```

### 🧱 Стек
Используй `ArrayDeque` вместо `Stack` — быстрее и современнее.

### 📦 Массивы
`IntArray(n) { idx -> ... }` часто компактнее ручного цикла с индексами.

### 🧵 Строки
`StringBuilder` эффективнее конкатенации через `+` в циклах.

### 📈 Монотонный стек
Кладём индекс в стек. Ждём элемент, который «ломает» монотонность, — обрабатываем.

### 📋 Условие задачи
В KDoc первой строкой — краткое описание на русском.
