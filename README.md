# newAlgo

Решения задач с [LeetCode](https://leetcode.com/) на Kotlin.

## Структура

```
src/main/kotlin/
├── Main.kt                — входная точка
└── topics/
    ├── arrays.kt          — задачи на массивы
    ├── strings.kt         — задачи на строки
    ├── stack.kt           — задачи на стек
    └── math.kt            — математические задачи
```

Kotlin 2.2.0, JVM 21.

---

## 💡 TIPS

| № | Совет |
|---|-------|
| 1 | **Запуск** — впиши вызов в `main()` и выполни `./gradlew run` |
| 2 | **Новая задача** — добавляй функцию в `topics/*.kt` по теме |
| 3 | **Тесты** — `./gradlew test` |
| 4 | **Стек** — используй `ArrayDeque` вместо `Stack` |
| 5 | **Массивы** — `IntArray(n) { idx -> ... }` компактнее цикла |
| 6 | **Строки** — `StringBuilder` быстрее `+` в циклах |
| 7 | **Монотонный стек** — клади индекс, жди элемент, ломающий монотонность |
| 8 | **Условие** — в KDoc краткое описание задачи на русском |
| 9 | **Ссылка** — в KDoc ссылка на оригинал на LeetCode |
