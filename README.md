# newAlgo

Решения задач с [LeetCode](https://leetcode.com/) на Kotlin.

## Структура

```
src/main/kotlin/
├── Main.kt                — входная точка
└── topics/
    ├── arrays.kt          — задачи на массивы
    ├── strings.kt         — задачи на строки
    ├── stack.kt           — задачи на стек
    └── math.kt            — математические задачи
```

Kotlin 2.2.0, JVM 21.

---

## 💡 TIPS

| № | Совет |
|---|-------|
| 1 | **Запуск решения** — впиши вызов функции в `main()` в `Main.kt` и выполни `./gradlew run` |
| 2 | **Новая задача** — создай функцию в соответствующем `topics/*.kt` файле |
| 3 | **Тесты** — если добавил тест в `src/test/kotlin/`, запусти `./gradlew test` |
| 4 | **Стек** — в Kotlin используй `ArrayDeque` вместо `Stack` (быстрее, современнее) |
| 5 | **Массивы** — `IntArray(n) { idx -> ... }` часто удобнее цикла с индексами |
| 6 | **Строки** — `StringBuilder` эффективнее конкатенации `+` в циклах |
| 7 | **Паттерн** — монотонный стек: кладём индекс, пока не встретим элемент, "ломающий" монотонность |
| 8 | **Подсказка к задаче** — в KDoc первой строкой указано краткое условие на русском |
| 9 | **Ссылка** — в KDoc всегда есть ссылка на оригинал задачи на LeetCode |
